```
┌─────────────────────────────────────────────────────────────┐
│              HERMES TIKTOK FACTORY                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. RESEARCH ──→ Scans trending topics, finds angles         │
│       ↓                                                     │
│  2. SCRIPT ────→ Writes engaging script per slideshow       │
│       ↓                                                     │

│  5. QUEUE ─────→ Prepares 10 slideshows ready for review    │

```

### Time Comparison

| Task | Manual (10 slideshows) | Hermes Automated |
|------|----------------------|-----------------|
| Topic research | 8-10 hours | **Minutes** |
| Script writing | 10-12 hours | **Seconds** per slideshow |
| Slide design | 10-12 hours | **Automated** |
| Formatting | 2-4 hours | **Automatic** |
| **Total** | **30-40 hours** | **1-2 hours (review only)** |

What used to fill a work week now takes about the same time as watching a movie. That opens room to:

- Scale output without adding time
- Spend more time on creative direction
- Use the same pipeline for Reels, YouTube Shorts
- Test different scripts, formats, topics

### What Hermes Needs

To build your own TikTok factory:

- Research skill for web scraping and trend analysis
- Script skill for short-form content
- Image generation or curated stock images
- Video assembly via FFmpeg or similar
- Formatting for 9:16 aspect ratio

---

### Setup

1. Install Hermes with image processing and video assembly capabilities
2. Create the research skill for topics, sources, angle extraction
3. Build the script engine for short-form attention spans
4. Set up slide generation via Canva API, Python-pptx, or image compositing
5. Add video assembly with FFmpeg
6. Configure batch processing via cron for 10 slideshows at a time
7. Create a review workflow with a folder or dashboard
8. Post manually or connect to TikTok API
