### How the Bypass Works

Instead of paying for the official X API, Hermes uses alternative methods:

- Browser-based interactions that mimic human behavior
- Third-party bridges that offer X posting without direct API fees
- SMS or notification-based posting via mobile workarounds
- Cross-posting from other platforms with free X access

The exact method changes over time. The Hermes community shares new approaches as they appear.

### The Roast Bot

- Random posts on a cron schedule throughout the day
- Real users find and interact with the bot
- When someone engages, Hermes roasts them back
- Roast bots tend to spread on their own

One user went back and forth without realizing they were talking to an AI.

### Setup

1. Install Hermes with web automation capabilities
2. Check the community Discord for current working X workarounds
3. Create the posting skill for roast style, topics, and frequency
4. Set up a cron schedule with randomized posting times
5. Add reply capability to monitor mentions
6. Set boundaries so the roast stays funny, not offensive
7. Review engagement and refine the approach

Platform terms of service vary. Always check what is allowed.
