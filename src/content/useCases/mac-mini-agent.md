### The Setup Breakdown

| Component | Cost |
|-----------|------|
| Mac Mini (already owned) | **$0** (sunk cost) |
| Electricity | **~$5/month** |
| Internet | **~$5/month** (shared) |
| OpenRouter API (Minimax + Sonnet) | **~$10/month** |
| Domain/DDNS | **~$1/month** |
| **Total** | **~$21/month** |

### What 18 Cron Jobs Look Like

A production agent running 18 cron jobs is doing something every ~80 minutes on average. Typical jobs include:

- **Market monitoring** — Check prices, news, and sentiment every hour
- **Content publishing** — Blog posts, social media, newsletters
- **Data backup** — Memory and context snapshots
- **Email processing** — Inbox zero automation
- **Research rounds** — Topic exploration and summarization
- **Learning reviews** — Self-reflection and strategy adjustment

### The Structured Context System

The most sophisticated part of this setup is the **structured context system** that makes every session smarter than the last. Instead of flat memory, it uses:

- **Hierarchical memory** — Recent context, weekly summaries, monthly archives
- **Cross-session knowledge** — What was learned in one session persists to the next
- **Skill refinement** — Skills are updated based on what worked
- **Performance metrics** — Tracking what Hermes is doing and how well

This means the agent on day 60 is significantly more capable than the agent on day 1 — not because the model changed, but because its context system accumulated and organized knowledge effectively.

### Mac Mini as an Agent Server

The Mac Mini is an underrated platform for running AI agents:

- **Silent operation** — No fan noise, runs 24/7 in a corner
- **Low power** — Idles at ~7W, under load ~30W
- **Reliable** — MacOS is stable, rarely crashes
- **Good performance** — M-series chips handle concurrent workloads well
- **Built-in UPS** — Laptop-style battery backup on power loss

## How You Can Do This Too

1. **Get a Mac Mini** — M1 or newer, can be used/refurbished
2. **Install Hermes** — Standard installation
3. **Plan your cron jobs** — Start with 2-3, expand as you see what works
4. **Build scripts incrementally** — Each script solves one problem
5. **Develop custom skills** — Create skills for your most common tasks
6. **Implement structured context** — Design a memory hierarchy that grows with use
7. **Track costs** — OpenRouter dashboard + electricity monitoring
8. **Let it run** — The system improves with time

The $21/month figure proves that a 24/7 production AI agent is within reach of anyone with a spare Mac Mini and a willingness to set it up properly.
