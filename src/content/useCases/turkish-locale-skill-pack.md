## Hermes in Any Language

Most AI tools are English-first. @erhnysr built a complete Turkish locale skill pack from scratch — real-time market data in Turkish Lira, news from Hürriyet, Bloomberg HT, and NTV, daily PNG briefing cards delivered to Telegram, all without a single external API key.

### What's in the Pack

| Component | Purpose |
|-----------|---------|
| Market data in TRY | Real-time stock/exchange rates in Turkish Lira |
| Turkish news sources | Hürriyet, Bloomberg HT, NTV |
| PNG briefing cards | Daily visual summary |
| Telegram cron | Automated daily delivery |
| Zero external API keys | Uses only free/hermes built-in tools |

### Why This Matters

This demonstrates that Hermes isn't tied to English or US-centric workflows. Anyone in any country can build a locale-specific agent that works with local data sources, currencies, and news outlets — without paying for premium API access.

## How You Can Do This Too

1. **Identify your locale** — Language, currency, news sources
2. **Find accessible sources** — Free RSS feeds, public APIs, or web scraping
3. **Build the data skill** — Market rates, news aggregation
4. **Create the briefing format** — PNG, PDF, or text digest
5. **Set up Telegram delivery** — Cron job with message formatting
6. **Iterate** — Add more sources and formats over time
